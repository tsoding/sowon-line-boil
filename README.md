# Font Line Boil Proof of Concept

Implemented for [Sowon](https://github.com/tsoding/sowon), but I'm not sure how to integrate it yet. An on going experiment.

## Dependencies

- [FreeType](https://freetype.org/)

## Build

```console
$ make
$ ./main ./ComicMono-Bold.ttf
$ feh digits.png
```
